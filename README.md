# 🌟 Proyecto Beneficiarios

Sistema para la gestión de beneficiarios desarrollado con HTML, CSS y JavaScript.

---

## 📋 Descripción

Este proyecto permite:

- Registrar beneficiarios.
- Consultar información.
- Editar registros.
- Eliminar registros.

---

## 🛠 Tecnologías

- HTML5
- CSS3
- JavaScript

---

## 🚀 Cómo ejecutar el proyecto

1. Descargar el repositorio.
2. Abrir el archivo `index.html`.
3. Utilizar la aplicación desde el navegador.
